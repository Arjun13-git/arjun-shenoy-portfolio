# Motion Guidelines

Version 1.0

---

# Philosophy

Motion should communicate state, hierarchy, and continuity.

Animations must never exist purely for decoration.

---

# Keywords

Fluid

Natural

Scientific

Calm

Responsive

Purposeful

---

# Animation Duration

Fast

150ms

Medium

300ms

Slow

500ms

Background

Continuous

---

# Easing

Use ease-out for entrances.

Use ease-in-out for transitions.

Avoid bounce animations.
